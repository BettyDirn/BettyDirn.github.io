geojson格式

geojson格式

概念

  • GeoJSON是一种用于编码各种地理数据结构的格式。GeoJSON对象可以表示几何，要素或要素集合。GeoJSON支持以下几何类型：Point，LineString，Polygon，MultiPoint，MultiLineString，MultiPolygon和GeometryCollection。GeoJSON中的功能包含几何对象和其他属性，功能集合表示功能列表。

  • 完整的GeoJSON数据结构始终是一个对象（以JSON术语表示）。在GeoJSON中，对象由一组名称/值对组成 - 也称为成员。对于每个成员，名称始终是一个字符串。构件值不是字符串，数字，对象，数组或文字中的一个：true，false，和 null。数组由元素组成，其中每个元素是如上所述的值。

GeoJSON特征集合

{ "type": "FeatureCollection",
"features": [
  { "type": "Feature",
    "geometry": {"type": "Point", "coordinates": [102.0, 0.5]},
    "properties": {"prop0": "value0"}
    },
  { "type": "Feature",
    "geometry": {
      "type": "LineString",
      "coordinates": [
        [102.0, 0.0], [103.0, 1.0], [104.0, 0.0], [105.0, 1.0]
        ]
      },
    "properties": {
      "prop0": "value0",
      "prop1": 0.0
      }
    },
  { "type": "Feature",
     "geometry": {
       "type": "Polygon",
       "coordinates": [
         [ [100.0, 0.0], [101.0, 0.0], [101.0, 1.0],
           [100.0, 1.0], [100.0, 0.0] ]
         ]
     },
     "properties": {
       "prop0": "value0",
       "prop1": {"this": "that"}
       }
     }
   ]
 }

几何示例

点：点坐标为x，y顺序（东经，北向为投影坐标，经度，纬度为地理坐标）。

{ "type": "Point", "coordinates": [100.0, 0.0] }

线串：LineString的坐标是一个位置数组。

{ "type": "LineString",
   "coordinates": [ [100.0, 0.0], [101.0, 1.0] ]
}

多边形：Polygon的坐标是LinearRing坐标数组的数组。数组中的第一个元素代表外环。任何后续元素代表内环（或孔）。
没有孔： 

{ "type": "Polygon",
"coordinates": [
  [ [100.0, 0.0], [101.0, 0.0], [101.0, 1.0], [100.0, 1.0], [100.0, 0.0] ]
  ]
}

带孔：

{ "type": "Polygon",
"coordinates": [
  [ [100.0, 0.0], [101.0, 0.0], [101.0, 1.0], [100.0, 1.0], [100.0, 0.0] ],
  [ [100.2, 0.2], [100.8, 0.2], [100.8, 0.8], [100.2, 0.8], [100.2, 0.2] ]
  ]
}

多点：MultiPoint的坐标是一系列坐标点。

{ "type": "MultiPoint",
"coordinates": [ [100.0, 0.0], [101.0, 1.0] ]
}

多线：MultiLineString的坐标是LineString坐标数组的数组。

{ "type": "MultiLineString",
"coordinates": [
    [ [100.0, 0.0], [101.0, 1.0] ],
    [ [102.0, 2.0], [103.0, 3.0] ]
  ]
}

多面：MultiPolygon的坐标是多边形坐标数组的数组。

{ "type": "MultiPolygon",
"coordinates": [
  [[[102.0, 2.0], [103.0, 2.0], [103.0, 3.0], [102.0, 3.0], [102.0, 2.0]]],
  [[[100.0, 0.0], [101.0, 0.0], [101.0, 1.0], [100.0, 1.0], [100.0, 0.0]],
   [[100.2, 0.2], [100.8, 0.2], [100.8, 0.8], [100.2, 0.8], [100.2, 0.2]]]
  ]
}

Geometry集合：GeometryCollection的几何数组中的每个元素都是上述几何对象之一。

{ "type": "GeometryCollection",
"geometries": [
  { "type": "Point",
    "coordinates": [100.0, 0.0]
    },
  { "type": "LineString",
    "coordinates": [ [101.0, 0.0], [102.0, 1.0] ]
    }
 ]
}
